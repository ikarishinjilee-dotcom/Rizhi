# 日值前端 MVP 验收清单

这份清单用于验收当前 Vue + IndexedDB 本地版是否已经达到“可以稳定试用”的程度。

使用方式：

- 验收时按顺序逐项检查。
- 通过的项标记为 `[x]`。
- 没通过的项保留 `[ ]`，并在“问题记录”里补充现象、页面、复现步骤。
- 涉及钱、账户余额、交易流水、删除操作的问题优先级最高。

## 0. 验收环境

- [ ] 能正常启动开发服务。
- [ ] 页面刷新后数据仍然保留。
- [ ] 控制台没有持续刷屏的错误。
- [ ] 页面没有明显白屏、错位、内容遮挡。
- [ ] 当前浏览器缩放 100% 时主要页面可正常使用。

建议命令：

```bash
npm.cmd run dev
```

## 1. 全局交互和安全

### 1.1 表单校验

- [ ] 必填项为空时，页面上显示明确错误提示。
- [ ] 金额为空、为 0、为负数或非法字符时，不能保存。
- [ ] 日期为空或格式异常时，不能保存。
- [ ] 错误提示不会导致左右输入框高度严重错位。
- [ ] 保存成功后，不会误弹“未保存提醒”。

### 1.2 日期选择

- [ ] 所有日期输入框点击后能打开日期选择器。
- [ ] 日期选择器层级高于当前弹窗。
- [ ] 点击日期后，输入框正确写入日期。
- [ ] 点击日期后，日期选择器自动关闭。
- [ ] 日期选择器不会被遮挡或跑到页面底层。

### 1.3 删除确认

- [ ] 所有删除按钮点击后都出现自定义确认弹窗。
- [ ] 确认弹窗文案能说明删除对象和影响。
- [ ] 点击取消不会删除数据。
- [ ] 点击确认才执行删除。
- [ ] 有占用关系的数据删除失败时，页面显示明确提示。

### 1.4 弹窗层级

- [ ] 业务弹窗居中显示。
- [ ] 弹窗高度不会超过浏览器可视区域导致顶部或底部不可见。
- [ ] 弹窗内容过长时，内部滚动正常。
- [ ] 多层弹窗时，最上层弹窗可以点击。
- [ ] 关闭按钮、取消按钮、保存按钮都能正常操作。

## 2. 看板页

- [ ] 看板统计数字能正常展示。
- [ ] 净资产、总资产、总负债、本月收入、本月支出、预算剩余数值合理。
- [ ] 即将过保物品列表可点击进入资产详情。
- [ ] 即将还款账户列表可点击进入资金账户或账户详情。
- [ ] 闲置较久物品列表可点击进入资产详情。
- [ ] 最近交易记录可点击查看或定位到交易。
- [ ] 图表不遮挡、不溢出。
- [ ] 空数据状态有合理展示。

- [ ] 资产卡片展示名称、品牌、分类、金额、持有天数、日均成本、状态。
- [ ] 没有上传图片的资产显示系统占位图。
- [ ] 上传过图片的资产显示资产图片。
- [ ] 分类筛选能正常过滤。
- [ ] 状态筛选能正常过滤。
- [ ] 搜索能按名称、品牌、型号匹配。
- [ ] 分页或列表数量显示正确。

### 3.2 新增资产

- [ ] 点击新增资产打开居中弹窗。
- [ ] 可以填写资产名称、品牌、型号、分类、购入价、日期、付款账户等。
- [ ] 可以上传多张资产图片。
- [ ] 可以移除图片，且移除前有确认。

- [ ] 如果选择付款账户，记账页出现资产购买交易。

## 4. 资产详情页

### 4.1 基础信息

- [ ] 面包屑显示“物品资产 / 资产详情”。
- [ ] 点击面包屑上一层能返回资产列表。
- [ ] 标题和资产名称不重复冲突。
- [ ] 图片大图和缩略图逻辑正确。
- [ ] 没有上传图片时只显示一张默认小图。
- [ ] 多张图片时可以点击缩略图切换大图。
- [ ] 成本概览显示原始购入价、附加项合计、资产总成本、当前估值、已损益、日均成本。

### 4.2 编辑资产

- [ ] 点击编辑打开和新增资产同一套弹窗。
- [ ] 弹窗内已有资产信息可编辑。
- [ ] 修改购入价后，资产购买交易和账户余额同步变化。
- [ ] 修改付款账户后，旧账户回滚，新账户写入。
- [ ] 移除付款账户后，购买交易被删除，账户余额回滚。
- [ ] 已转让或已处置资产不能继续编辑会影响财务的内容。

### 4.3 资产状态

- [ ] 使用中资产可以标记闲置。
- [ ] 闲置资产可以恢复使用中。
- [ ] 使用中或闲置资产可以转让。
- [ ] 已转让资产不能再次转让。
- [ ] 已处置资产不能转让。
- [ ] 已转让资产只显示合理操作，例如撤销转让和删除。
- [ ] 撤销转让后，转让收入交易删除，账户余额回滚，资产恢复使用中。

- [ ] 点击添加附加项打开居中弹窗。
- [ ] 可以选择方向为支出。
- [ ] 支出型类型展示为配件、维修、保养、升级、耗材、其他。
- [ ] 可以填写名称、金额、日期、付款账户、备注。

- [ ] 如果不计入成本，资产总成本不增加。

### 5.2 新增收入型附加项

- [ ] 可以选择方向为收入。
- [ ] 收入型类型展示为配件转卖、共享回款、退款赔偿、残值回收、耗材回款、其他收入。

- [ ] 如果选择收款账户，记账页生成收入交易。

- [ ] 资产总成本不因收入型附加项减少。

### 5.3 编辑和查看附加项

- [ ] 点击查看能打开附加项详情弹窗。
- [ ] 点击编辑能编辑附加项内容。

- [ ] 修改附加项方向后，交易类型和账户影响正确重算。
- [ ] 修改附加项图片后，详情和列表展示同步更新。
- [ ] 删除附加项前有确认。
- [ ] 已转让或已处置资产不能新增或编辑附加项。

- [ ] 支出表单可以选择对应子分类。
- [ ] 收入表单可以选择一级分类。

### 6.3 资产关联

## 7. 资金账户页

- [ ] 近期还款提醒展示正确。

### 7.2 添加和编辑账户

- [ ] 点击添加账户打开居中弹窗。
- [ ] 可以选择资产账户或负债账户。

- [ ] 必填项为空时不能保存。
- [ ] 保存后账户出现在资金账户页。

- [ ] 转出账户和转入账户不能相同。

- [ ] 已有资金流水的账户不能删除。

- [ ] 可以导出本地数据。

- [ ] 导入后核心数据能正常显示。

## 10. 验收命令

普通前端改动：

```bash
npm.cmd test
npm.cmd run typecheck
```

涉及弹窗、删除、上传、日期选择：

```bash
npm.cmd run test:e2e
npm.cmd test
npm.cmd run typecheck
```

阶段性交付：

```bash
npm.cmd run test:e2e
npm.cmd test
npm.cmd run typecheck
npm.cmd run build
```
