### 2.4 E2E 测试

命令：

```bash
npm.cmd run test:e2e
```

用例结果：5 个用例均显示 `ok`。

覆盖内容：

- 资产/附加项弹窗保存后不误弹未保存提醒。
- 图片移除确认弹窗可点击。
- 资产删除必须确认。
- 记账页交易删除必须确认。
- 资金账户删除必须确认，且有流水账户不能删除。
- 分类有交易占用时不能删除。

发现问题：

- E2E 用例全部通过后，Playwright 命令没有正常退出，最终被命令超时终止。
- 已尝试把 `playwright.config.ts` 的 webServer 命令从 `npm.cmd run dev` 改为直接 `node ./node_modules/vite/bin/vite.js` 启动 Vite，但命令仍会在用例完成后挂住。
- 当前判断：功能用例本身通过，但 E2E runner 收尾存在工程问题，需要单独排查。

修复结果：

- 已新增 `scripts/run-e2e.mjs`。
- `npm.cmd run test:e2e` 改为由脚本手动启动 Vite、等待服务可访问、执行 Playwright、最后关闭 Vite 进程树。
- 复跑结果：5 个 E2E 用例通过，命令正常退出。
